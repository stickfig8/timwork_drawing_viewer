# 팀워크 도면 뷰어

개발자 : 김현규

## 실행방법

```bash
npm install
```

```bash
npm run dev
```

## 기술 스택

- React
- TypeScript
- Tailwind
- Shadcn
- Zustand
- eslint + prettier

## 구현 기능

1. 검색 사이드 바
   - 건물 / 공종 / 리전 / 리비전 / 검색어 기반 필터
   - 건물 선택 시 자동 공종 활성화/비활성화 처리
   - 조건 조합 기반 도면 리스트 생성
   - 검색결과 정렬 설정
   - 비교 가능 도면만 필터링 옵션 제공
   - 도면 탐색에 집중하기 위한 최소화 기능

2. 선택 도면 네비게이터
   - 현재 보고있는 도면 상세 정보 표시
   - 비교 중인 도면 목록 관리
   - 레이어 제거 / 이동 기능
   - 네비게이터 드래그, 최소화 기능
3. 도면 뷰어
   - 선택 도면이 없을 시 전체 도면 + 폴리곤 기반 건물 선택 인터렉션 제공
   - 선택 도면 표시
   - 드래그, 확대, 축소 지원
   - 키보드 단축키
     - `+` / `-` : 확대 축소
     - 0 : 줌 초기화
     - 방향키 위 아래: 비교중인 이전 / 다음 도면으로 이동
4. 미완성 기능
   - scale, x/y, rotation기반 정렬 완전 적용
   - 하위 요소 폴리곤 출력
   - 북마크
